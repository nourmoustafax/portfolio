Nour Portfolio Website - Deployment Notes

This is a static website portfolio.

Files:
- index.html
- styles.css
- assets/churn-dashboard.png

How to publish it publicly:
1. GitHub Pages:
   - Create a repository on GitHub.
   - Upload all files and folders exactly as they are.
   - Go to Settings > Pages.
   - Under Source, choose Deploy from a branch.
   - Select main branch and /root.
   - Save.

2. Netlify:
   - Drag and drop the whole folder into Netlify.

3. Vercel:
   - Import the folder through a GitHub repository and deploy.

Important:
- Keep the assets folder in the same structure.
- index.html must stay at the root.
